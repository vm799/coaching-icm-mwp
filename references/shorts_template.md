# Shorts Template — YouTube Shorts / Reels / TikTok (60 sec)

Stage 04 produces shorts specs from long-form scripts.
Source: best moments from teaching or enterprise scripts.

---

# [Topic] — Shorts Package

## Identify Moments (From Script)

Best short-form moments come from:
- Opening hook (first 60 sec of full video — already hook-shaped)
- Analogy reveal ("X is like Y because...")
- Surprising data point ("Most people don't realize that...")
- Soundbite / catchphrase (enterprise scripts — already quotable)
- Demo reveal moment ("And here's what happens when we run this...")

**Target: 3-5 shorts per topic**

---

## Short Video Structure (60 sec)

```
0:00–0:05  HOOK — Stop the scroll
0:05–0:45  VALUE — Deliver the insight fast
0:45–0:55  PROOF / EXAMPLE — Show it works
0:55–1:00  CTA — Watch the full video
```

**Rule:** Hook in 5 seconds or lose them. Don't warm up. Start with the punchline.

## Short Templates

### Template A: Surprising Claim
```
0:00-0:05  "[Surprising/counterintuitive claim]"
            (e.g., "More context doesn't mean better AI answers.")

0:05-0:40  Explain WHY — 2-3 sentences + one visual/demo
            (e.g., "Here's what actually happens when you overload context...")

0:40-0:55  Proof: "We tested this. Result: [metric]"

0:55-1:00  CTA: "Full explanation in the link above"
```

### Template B: Analogy Drop
```
0:00-0:05  "[X] is like [Y]."
            (e.g., "Tokens are like attention. Attention is expensive.")

0:05-0:40  Unpack the analogy
            (e.g., "When you spend attention on the wrong thing, you miss what matters...")

0:40-0:55  Connect to real impact
            (e.g., "In AI systems, this costs money AND accuracy.")

0:55-1:00  CTA: "Here's how to fix it — full video above"
```

### Template C: Quick Demo
```
0:00-0:05  "Watch what happens when I [action]"

0:05-0:40  Run demo — no explanation yet, let them see it
            Fast cuts, zoom on key output

0:40-0:55  "That [result] happened because [mechanism in 1 sentence]"

0:55-1:00  CTA: "Full walkthrough in the video above"
```

### Template D: Hot Take / Opinion
```
0:00-0:05  "[Strong opinion about the topic]"
            (e.g., "Most teams are solving the wrong AI problem.")

0:05-0:40  "Here's what I mean: [evidence + argument]"

0:40-0:55  "The better approach: [brief alternative]"

0:55-1:00  CTA: "Full breakdown in the video"
```

### Template E: 3-Part Framework (Teaching)
```
0:00-0:05  "3 things you need to understand about [X]:"

0:05-0:20  "1. [Point 1 — one sentence]"
0:20-0:40  "2. [Point 2 — one sentence]"
0:40-0:55  "3. [Point 3 — one sentence]"

0:55-1:00  CTA: "Deep dive on all 3 — full video above"
```

## Captions / Text Overlays

Shorts are watched muted. Text on screen = critical.

**Always include:**
- Hook text as large on-screen text (0-5 sec)
- Key word/phrase as text overlay during body
- CTA as text (0:55-1:00)

**Format:**
- Font: bold sans-serif (e.g., Impact / Montserrat Bold)
- Color: white with dark shadow OR bright accent on dark background
- Size: readable at phone size
- Position: upper third or center (not bottom — covered by UI)

## Shorts Calendar (Per Topic)

Release 3-5 shorts per topic. Space them out:

| Release | Type | Timing |
|---------|------|--------|
| Short 1 | Hook / surprising claim | Day of YouTube publish |
| Short 2 | Analogy reveal | Day +3 |
| Short 3 | Demo moment | Day +7 |
| Short 4 | Soundbite (enterprise) | Day +14 |
| Short 5 | 3-part framework | Day +21 |

**Why spread out?** Each short re-triggers algorithm. Topic stays discoverable for 3-4 weeks instead of 1 day.

---

## Shorts → Long-Form Funnel

Short's only job: drive to long-form content.

**CTA options:**
- "Full 30-min breakdown in my YouTube — link above"
- "Watch the full demo — link in bio"
- "I cover this in depth at [channel name] — see link above"

**Track:** Use YouTube analytics to see which shorts drove long-form views.
Top-performing short = make more like it.

---

## Checklist Before Exporting Shorts

- [ ] Hook is in first 5 seconds (no warm-up)
- [ ] 9:16 format (vertical) — NOT widescreen
- [ ] Captions burned in (muted viewers can follow)
- [ ] CTA is clear
- [ ] No long silences (edit dead time)
- [ ] Audio: clean, normalized (no pops or background noise)
- [ ] Text: readable at small size
- [ ] Length: under 60 sec (Shorts), under 90 sec (Reels)
