# Publishing Checklist — Create Once, Publish Everywhere

Run this checklist for each topic. One script → every channel.

---

## Pre-Production (Before Recording)

- [ ] `setup/questionnaire.md` filled and saved to `output/[topic-slug]/`
- [ ] `discovery_brief.md` reviewed and approved
- [ ] `research_output.md` reviewed and approved (5+ sources, data backed)
- [ ] `teaching_script.md` reviewed — can read aloud, no stumbling
- [ ] `enterprise_script.md` reviewed — soundbites marked, ROI clear
- [ ] Demo code written, tested, and committed to repo (if code topic)
- [ ] Deck outline reviewed — slides match [SLIDE: ...] markers in script

---

## Recording Day

- [ ] Video recorded (30 min, full script)
- [ ] Audio clean (no background noise, normalized levels)
- [ ] Screen + face cam working (if show-and-tell / demo)
- [ ] Demo ran successfully (or backup screenshots captured)
- [ ] Raw file saved to local drive + cloud backup

---

## Post-Production (Video)

- [ ] Video edited (dead time cut, transitions clean)
- [ ] Captions exported (SRT or VTT file)
- [ ] 3-5 short clips identified and exported (60 sec each)
- [ ] Thumbnail created (1280x720px, readable at small size)
- [ ] Teaching version: warm tone, analogy visual
- [ ] Enterprise version: bold data or soundbite visual

## Quality Gates (Before Each Publish)

### YouTube
- [ ] Video plays end-to-end without issues
- [ ] Title includes keyword (50-60 chars)
- [ ] Timestamps accurate (test: click each, lands in right place)
- [ ] Captions accurate (spot check 3 sections)
- [ ] Thumbnail readable at 150px width

### Blog
- [ ] No "script-isms" remaining ([SLIDE:...], "Hey there", [PAUSE:...])
- [ ] All citations linked
- [ ] Images have alt text
- [ ] CTA is clear (one link, one action)
- [ ] YouTube video embedded

### Email
- [ ] Subject line under 50 chars
- [ ] Sends correctly (test to yourself first)
- [ ] Mobile renders cleanly (test on phone)
- [ ] Unsubscribe link working

### LinkedIn
- [ ] First 3 lines hook without needing "See more"
- [ ] Image attached
- [ ] One CTA only

### Stage Talk
- [ ] Deck uploaded to AV team 48h before event
- [ ] Backup slides on USB + phone
- [ ] Speaker notes printed (backup for teleprompter failure)
- [ ] Demo tested on event machine (not just your laptop)

## Analytics Review (Monthly)

Review once per month. Update metadata on underperforming content.

| Metric | Good | Review Needed |
|--------|------|---------------|
| YouTube click-through rate | >6% | <4% → new thumbnail |
| YouTube average view duration | >50% | <30% → fix pacing in future |
| Blog read rate | >50% | <30% → fix headline or structure |
| Email open rate | >30% | <20% → test new subject lines |
| LinkedIn article views | >500 | <100 → boost with post promotion |
| Short-form → long-form click | >2% | <1% → fix CTA |

---

## Repurpose Trigger: When To Go Back

High-performing content (any metric 2x average) → repurpose:
- New LinkedIn post with different angle
- New Twitter thread with different hook
- New Short (different clip from same video)
- New email to different list segment

Low-performing content → investigate before repurposing:
- Wrong audience? (change distribution)
- Wrong hook? (test new title/thumbnail)
- Wrong time? (re-publish at better time)
- Wrong format? (works better as article than video?)
